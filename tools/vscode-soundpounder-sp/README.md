# SoundPounder .sp Syntax (Local)

This is a local VS Code extension for SoundPounder `.sp` song files.

## Install (no Node/vsce required)

```bash
./tools/vscode-soundpounder-sp/install.sh
```

Then reload VS Code.

## Notes

- `code --install-extension` only accepts an extension ID or a `.vsix` file; it does not install from a folder.
